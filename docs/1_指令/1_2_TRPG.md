## .coc7 / #coc7
生成COC7的属性表。#coc7为娱乐用的每日更新，.coc7为完全随机。

## .sc
理智检定。格式：.sc 成功表达式/失败表达式 [当前理智值]

如果省略当前理智值，则根据今日的#coc7结果确定初始理智。

## .en
角色技能成长鉴定。格式：.en 属性/技能名 属性/技能值

如：.en 躺平 50

## .ra / .rc
理智检定，ra房规，rc规则书。格式：.ra/.rc 属性/技能名 属性/技能值

## .r / .rd / .rh / .rc
投掷骰子。

指令格式，括号内为可选参数及默认值:

* .r 原因 (1D100)
  
* .r 表达式 原因
  
* .rd (原因)

举例：
```text
.r
.r 测试
.rd
.rd 测试
.r 1D100 测试
.r 1D(100+100) 测试
.r 50*50 测试
```

## .li / .ti
生成疯狂症状。

## .bg
生成人物背景。

## .name
生成随机人名。

可选语言：英、中、日、英汉。
